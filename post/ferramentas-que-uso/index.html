Ferramentas Que Uso e recomendo - Fri, Dec 23, 2022

O que eu uso e o que não uso no dia a dia

Básico

Sistema operacional

Atualmente estou usando o Windows juntamente do Fedora com o gestor de janelas i3wm (nos últimos anos tenho preferido usar gestores de janelas por serem mais leves que interfaces gráficas).

Programação

Normalmente uso o Vim e o VScode, as vezes. Porem, tento priorizar o uso do vim.

Navegador

Quando uso o Windows uso o Opera GX já quando uso o Fedora utilizo o Brave.

O que eu evito usar

Software Proprietário

Eu sempre prefiro usar software livre pela sua:

  • Liberdade de executar o programa qualquer que for o propósito;

  • Liberdade de estudar como o programa funciona e adaptá-lo para as suas necessidades, sendo liberado o acesso ao código-fonte;

  • Liberdade de redistribuir cópias de modo que você possa ajudar outros usuários;

  • Liberdade de aperfeiçoar o programa, e liberar os seus aperfeiçoamentos, de modo que toda a comunidade se beneficie, sendo o acesso ao código-fonte um pré-requisito para esta liberdade. Porem utilizo softwares proprietários no ultimo caso quando não há alternativa aberta.

Interfaces Gráficas

Prefiro usar Wms pois não só são mais leves como mais práticas de usar.
