## O que eu evito usar
### Software Proprietário
Eu sempre prefiro usar software livre pela sua: 
+ Liberdade de executar o programa qualquer que for o propósito;

+ Liberdade de estudar como o programa funciona e adaptá-lo para as suas necessidades, sendo liberado o acesso ao código-fonte;

+ Liberdade de redistribuir cópias de modo que você possa ajudar outros usuários;

+ Liberdade de aperfeiçoar o programa, e liberar os seus aperfeiçoamentos, de modo que toda a comunidade se beneficie, sendo o acesso ao código-fonte um pré-requisito para esta liberdade.
Porem utilizo softwares proprietários no ultimo caso quando não há alternativa aberta.
### Interfaces Gráficas

Prefiro usar Wms pois não só são mais leves como mais práticas de usar.
